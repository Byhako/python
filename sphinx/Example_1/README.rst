Calculator
**********

Whats is this code for
----------------------

This code is a simple **calculator** which can only perform the addition or the
subtraction of two numbers.


Code requeriments
-----------------

None

Execution
---------

Run code like this::

  $ python calculator.py --a= --b= --operation=addition
